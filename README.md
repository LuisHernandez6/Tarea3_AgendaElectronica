### Tarea3_AgendaElectronica
### Luis Alonso Hernandez Perdomo / 2023-1804

Agenda electrónica la cual permite guardar contactos con datos como: nombre, apellido, correo electrónico, teléfono, etc. Además de guardar contactos, se pueden actualizar o eliminar estos. 

Los contactos son alojados en una base de datos **SQL Server** con la cual la aplicación se comunica mediante un **CRUD**, por lo tanto, es necesario tener **SQL Server** instalado y el **servicio del motor de SQL Server** corriendo. La base de datos se crea automáticamente al momento de correr la aplicación.
